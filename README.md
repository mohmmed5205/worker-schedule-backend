# جدول العمال الأسبوعي

مشروع كامل بواجهة React وباكند Express. الأدمن يدخل ويسوي تشيك لأيام دوام العمال، والعامل يدخل برقم الدخول ويشوف أيامه وحالة التشيك الخاصة فيه.

## المميزات

- واجهة عربية RTL.
- تسجيل دخول للأدمن والعمال.
- جدول سنة كاملة يبدأ من السبت 2026-06-27.
- حذف يوم الجمعة من أيام العمل.
- كل عامل له أسبوع كامل من السبت إلى الخميس.
- كل دورة = 8 أسابيع، والدورة الأخيرة تكمل باقي السنة.
- الأدمن يقدر يفلتر حسب الدورة أو العامل.
- العامل يشوف أيامه فقط.
- التخزين في ملف JSON محلي داخل السيرفر: `server/data/db.json`.

## التشغيل محليًا

ثبت الحزم:

```bash
npm install
npm install --prefix server
npm install --prefix client
```

أنشئ ملف إعدادات محلي باسم `server/.env`، وضع فيه رقم دخول الأدمن وأرقام العمال. لا ترفع هذا الملف إلى GitHub.

بعدها شغل الواجهة والباكند معًا:

```bash
npm run dev
```

افتح الواجهة:

```text
http://localhost:5173
```

الباكند يعمل على:

```text
http://localhost:4000
```

## ملفات مهمة

- `client`: واجهة React.
- `server`: باكند Express.
- `server/src/workers.js`: أسماء العمال وربط كل عامل بمتغير بيئة.
- `server/src/schedule.js`: توليد الجدول السنوي بدون يوم الجمعة.
- `server/data/db.json`: يتولد تلقائيًا عند تشغيل السيرفر، ولا يتم رفعه للريبو.

## ملاحظات مهمة

لا ترفع ملف `.env` إلى GitHub. خله محليًا أو ضعه كمتغيرات بيئة في خدمة الاستضافة.
